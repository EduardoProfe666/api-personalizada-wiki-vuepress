# Linea

## Provisional

<b>`Linea`</b>: Permitirá generar una línea, pudiendo ser personalizada con el grosor, color, orientación y el tipo de línea.

![a](/gifs/linea.gif)

> Constructores:
>
> ```java
> Linea lin = new Linea(Linea.ORIENTACION_VERTICAL, Linea.LINEA_DISCONTINUA);
> ```
