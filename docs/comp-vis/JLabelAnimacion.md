# JLabelAnimacion

## Provisional

<b>`JLabelAnimacion`</b>: Permitirá la modelación de un JLabel con texto animado.

![a](/gifs/label-animacion.gif)

> Constructores:
>
> ```java
> JLabelAnimacion lbl = new JLabelAnimacion("HOLA", -1, -1, -1);
> ```
