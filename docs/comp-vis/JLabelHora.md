# JLabelHora

## Provisional

<b>`JLabelHora`</b>: Permitirá la modelación de un reloj digital, con distintos formatos de hora.

![a](/gifs/reloj.gif)

> Constructores:
>
> ```java
> JLabelHora reloj = new JLabelHora(JLabelHora.FORMATO_PREDETERMINADO);
> ```
